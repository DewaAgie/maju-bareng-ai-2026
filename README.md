## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Before you begin, ensure you have the following installed:
* [Node.js](https://nodejs.org/) (v24.x or higher)
* [npm](https://www.npmjs.com/) (v12.x or higher)
* [Git](https://git-scm.com/)
* [Google AI Studio](https://aistudio.google.com/) for get [API Key](https://aistudio.google.com/app/apikey)

---

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/DewaAgie/maju-bareng-ai-2026.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd maju-bareng-ai-2026
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Add .env file**
   ```bash
   touch .env
   ```

5. **Add API Key to .env file**
   ```bash
   GEMINI_API_KEY=your-api-key-here
   ```

6. **Run the project**
   ```bash
   npm run dev
   ```

7. **Open the project on your browser**
   ```bash
   http://localhost:3000
   ```
